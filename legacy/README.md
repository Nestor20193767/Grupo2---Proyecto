# Código legado

Esta carpeta conserva versiones previas para trazabilidad, pero no forma parte de la ruta principal de ejecución.

- `legacy/app/`: versiones antiguas de la interfaz.
- `legacy/mit_bih/`: análisis iniciales basados en MIT-BIH, diferentes del pipeline final PhysioNet.

No mezcle métricas, frecuencia de muestreo ni longitud de ventana del código MIT-BIH con el experimento final.
