# Experimentos MIT-BIH archivados

El repositorio inicial contiene notebooks y scripts de una etapa exploratoria con MIT-BIH. Se archivan aquí porque el trabajo final y la aplicación desplegada usan PhysioNet ECG-Arrhythmia, 500 Hz y ventanas de 325 muestras.

Ejecute `scripts/migrate_repository.py` sobre el repositorio original para copiar esos archivos sin perder historial.
